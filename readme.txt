This is the official plugin for POLi Payments in WooCommerce.

== Description ==


Adding the POLi Payments plugin to WooCommerce will allow your customers in Australia and New Zealand to pay you straight from their internet bank account.

Head to https://signup.apac.paywithpoli.com/ to get a merchant account today!


== Installation ==


1. Upload the folder `poli-payments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your settings in the payment gateways admin panel in Woocommerce.
4. Click on the POLi Payments link at the top of the page.
Be sure you enter your Merchant Code, Authentication Code and enable POLi Payments.
